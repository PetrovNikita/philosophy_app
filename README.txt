Для запуска приложения выполнить в консоли в директории проекта команды:
    npm install
    npn start

Функционал проекта:
    Авторизованным пользователям доступны тексты и комментарии к ним, также можно оставлять свои комментарии.
    В блоке навигации можно выбрать сначала категорию, потом из раскрывшегося списка выбрать текст из категории.
    Реализована аутентификация с использованием cookie.
    Для просмотра текстов и комментариев к ним необходимо зарегистроваться, заполнив форму регистрации или залогиниться
        , заполнив форму которая появляется при нажатии кнопки: enter в заголовке.
    В формах проверяется валидность введенных данных в полях при снятии фокуса, поле ввода телефона форматируется 
        и валидируется при каждом при вводе данных, поле логина валидируется в обоих формах посредством проверки наличия 
        (когда пользователь логинится) или отсутствия (при регистрации) введенного логина в базе
        , также при фокусировке на поле логина, оно автоматически заполняется последним введенными логином.
    На React переписаны: компоненты header и комментарии (вместе с формой ввода коммента).Комментарии отсортированы: вверху 
        показываются более поздние.
